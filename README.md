# Sports-Hall-Booking-System-by-Service-Portal-ServiceNow-
The Sports Hall Booking System is a custom-built ServiceNow solution developed using Service Portal to simplify and automate sports facility reservations. It enables users to view real-time slot availability, submit booking requests, and track approvals through an intuitive portal interface. The system automates approvals, notifications, and status updates using Flow Designer and scheduled jobs, ensuring a secure, efficient, and scalable booking experience
